## Instalación

Para instalar las dependencias del proyecto, ejecuta el siguiente comando:

```bash
npm install
```

## Ejecución

Para ejecutar el proyecto en un entorno de desarrollo, usa el siguiente comando:
```bash
npm run dev
```
